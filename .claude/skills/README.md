# Project skills

Six session-rigor skills vendored verbatim from **Iwo's Rigor Pack v1.0.0**
(https://www.iwoszapar.com/tools/rigor-pack, MIT-spirited free pack, blind-
benchmarked by its author; method and full results in the pack's benchmarks.md).

Vendored 2026-07-03 after a full review: every file read; no network calls,
no privilege moves, no review-bypass instructions; the installer's embedded
copies verified byte-identical to the loose files. Kept verbatim so future
upstream versions diff cleanly — put Sanction-specific guidance in AGENTS.md,
not in these files.

| Skill | One line |
|---|---|
| plan-gate | Written plan (goal/unknowns/success criteria/steps) before the first edit |
| adversarial-verify | Attack your own work before presenting it; verdicts, not vibes |
| live-state-truth | Verify against the live system; docs and memory are stale by default |
| scope-fence | Change exactly what was asked; flag adjacent issues, never drive-by fix |
| ruthless-editor | Cutting pass on all prose; every sentence earns its place |
| memory-hygiene | What deserves persisting, dated; recalled facts re-verified before use |
